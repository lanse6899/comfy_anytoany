# ComfyUI 任意映射节点 (anytoany)

一个ComfyUI自定义节点插件，允许将任何输入映射到任何其他输入。

PS：商用平台方使用请知会一声哈！

## 功能特性

- ✅ **任意映射**：支持将任何类型的值映射到任何其他类型的值
- ✅ **多种类型支持**：字符串、数字、通用类型
- ✅ **兼容所有ComfyUI版本**：使用标准ComfyUI API
- ✅ **简单易用**：直观的节点接口

## 节点列表

### 1. anytoany (任意映射)
- **输入A**：任意类型输入
- **输入B**：任意类型输入
- **输出**：两个输入值（输出A, 输出B）
- **用途**：最通用的映射节点，适用于所有类型

### 2. AnyEqualsAnyString (字符串映射)
- **输入A**：字符串
- **输入B**：字符串
- **输出**：两个字符串值
- **用途**：专门用于字符串映射，例如："小猫" = "3" 或 "1" = "B"

### 3. AnyEqualsAnyNumber (数字映射)
- **输入A**：浮点数
- **输入B**：浮点数
- **输出**：两个浮点数值
- **用途**：专门用于数字映射

### 4. AnyEqualsAnyUniversal (通用映射)
- **输入A类型**：STRING/FLOAT/INT
- **输入A值**：字符串（会根据类型自动转换）
- **输入B类型**：STRING/FLOAT/INT
- **输入B值**：字符串（会根据类型自动转换）
- **输出**：转换后的值和类型信息
- **用途**：最灵活的类型转换和映射节点

## 安装方法

1. 将整个文件夹复制到ComfyUI的 `custom_nodes` 目录
2. 重启ComfyUI
3. 在节点菜单中找到 "anytoany" 分类下的映射节点

## 使用示例

### 示例1：字符串映射
- 使用 **🔵BB 字符串映射 (String = String)** 节点
- 输入A: "小猫"
- 输入B: "3"
- 输出：两个字符串 "小猫" 和 "3"

### 示例2：数字映射
- 使用 **🔵BB 数字映射 (Number = Number)** 节点
- 输入A: 1.0
- 输入B: 2.0
- 输出：两个浮点数 1.0 和 2.0

### 示例3：通用映射
- 使用 **🔵BB 通用映射 (Universal)** 节点
- 输入A类型: STRING，值: "小猫"
- 输入B类型: INT，值: "3"
- 输出：对应的值和类型

### 示例4：任意映射（推荐）
- 使用 **🔵BB 任意映射 (Any = Any)** 节点（即anytoany节点）
- 输入A: "小猫"
- 输入B: "3"
- 输出：两个字符串 "小猫" 和 "3"

## 兼容性

- ✅ ComfyUI 所有版本
- ✅ Windows / Linux / macOS
- ✅ Python 3.8+

## 注意事项

- 节点会直接传递输入值，不做任何修改
- 类型转换仅在 **🔵BB 通用映射 (Universal)** 节点中进行
- 确保输入值符合目标类型的格式要求
